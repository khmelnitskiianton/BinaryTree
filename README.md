## Introduction

To start programm you need to use CMake and run program

1. Clone repository:

    '''c
    $ git clone
    '''

2. Compile CMake, write command in main directory in repo:

    '''c
    $ cmake .
    '''

3. Run CMake (compile program), write command in main directory in repo:

    '''c
    $ make
    '''

4. Run program:

    '''c
    $ ./proga
    '''
